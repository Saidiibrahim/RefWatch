# Core Module

This module contains shared functionality and components used across multiple features.

## Components

- **Components**: Reusable UI elements
  - Custom buttons
  - Event buttons
  - Score displays
  - Setting pickers
  - Team official selection

## Services

- **MatchStateService**: Manages match state and transitions
- **TimerService**: Handles timing-related functionality

## Guidelines

- Keep components generic and reusable
- Avoid feature-specific logic in core components
- Follow consistent naming and documentation patterns
- Services should be stateless when possible
