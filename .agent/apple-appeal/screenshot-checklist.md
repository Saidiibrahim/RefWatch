# Screenshot & Evidence Gathering Checklist

## Required Screenshots for Apple Appeal

Use this checklist to gather all necessary evidence to support your appeal.

## üì± Screenshot Requirements

### General Guidelines:
- Use actual Apple Watch hardware or high-quality simulator
- Ensure UI is clear and readable
- Capture full screen (no crops)
- Use light mode for consistency

- Name files descriptively (e.g., "workout-mode-selection.png")
